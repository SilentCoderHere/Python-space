# Python Space

A personal collection of Python programs, experiments, and learning
projects.

## About

Python Space is a growing repository where I explore Python through
small projects, practice scripts, and simple applications. The goal is
to learn by building, keep code clean, and improve step by step.

## What You'll Find

-   Beginner to intermediate Python programs
-   Practice problems and solutions
-   Small utilities and experiments
-   Examples focused on clarity and understanding

## Tech Stack

-   Python 3

## Getting Started

1.  Clone the repository:

    ``` bash
    git clone https://github.com/jesterdotcom/python-space.git
    ```

2.  Navigate into the project folder:

    ``` bash
    cd python-space
    ```

3.  Run any Python file:

    ``` bash
    python filename.py
    ```
